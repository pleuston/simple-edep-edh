# simple-edep-edh — EDH data collection

Open-data collection consumed by the [simple-edep](https://github.com/pleuston/simple-edep)
app (served via jsDelivr). Contains the **Epigraphic Database Heidelberg (EDH)** export:

- `records/HD*.xml` — ~81,883 EpiDoc inscription records (HD000001–HD082828).
- `collections/edh/` — derived registers: `index.json`, `people.json`, `photos.json`,
  `geo.json`, `provinces.json`, `bibliography.json`, `manifest.json`.
